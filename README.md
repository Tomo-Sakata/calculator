# Calculator
This is a project from The Odin Project Foundations course.

## Objectives
The primary goals of this project were to:
- Build a functional calculator interface using HTML and CSS
- Practice layout techniques with Flexbox and CSS Grid
- Implement interactive behavior with JavaScript (event listeners, DOM manipulation)
- Handle user input and perform basic arithmetic operations
